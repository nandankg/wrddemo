Leadership portraits for the "Our Leadership / नेतृत्व" band (includes/leaders.php).

Drop a square JPG here named after each leader's slug and the band will use it
automatically — no code change needed. Until a file is present, a dignified
initials placeholder is shown instead.

Expected filenames:
  hemant-soren.jpg           — Shri Hemant Soren, Hon'ble Chief Minister, Jharkhand  (hero, right of hero text)
  hafizul-hassan.jpg         — Shri Hafizul Hassan, Hon'ble Minister, WRD            (hero, right of hero text)
  prashant-kumar.jpg         — Shri Prashant Kumar, Secretary, WRD                   (band below hero)
  additional-secretary.jpg   — Additional Secretary, WRD                             (band below hero)

Recommended: square crop, ~400x400 px, head-and-shoulders, neutral background.
The band displays the secretaries at 96x96 as circles and the hero portraits at
64x64 as rounded squares, so keep the face centred.
